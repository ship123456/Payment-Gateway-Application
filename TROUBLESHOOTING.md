## 1. Introduction

This guide provides solutions to common issues that users may encounter while using the Payment Gateway.

Before contacting support, review the troubleshooting steps provided in this document. Many common issues can be resolved by following the recommended solutions.

## 2. Login Issues

| Issue | Possible Solution |
|-------|-------------------|
| Unable to log in | Verify your email address and password. |
| Incorrect password | Use the **Forgot Password?** option to reset your password. |
| Account locked | Wait for the lockout period to expire or contact your system administrator. |
| Login page not loading | Refresh the page or try another supported browser. |

## 3. Payment Issues

| Issue | Possible Solution |
|-------|-------------------|
| Payment failed | Verify your payment details and try again. |
| Payment pending | Wait a few minutes and refresh the transaction status. |
| Incorrect payment amount | Cancel the transaction, if possible, and create a new payment with the correct amount. |
| Transaction not visible | Refresh the **Payment History** page or check again after a few minutes. |

## 4. Refund Issues

| Issue | Possible Solution |
|-------|-------------------|
| Refund request failed | Ensure the transaction is eligible for a refund before submitting the request. |
| Refund not received | Refund processing times vary depending on the payment method and financial institution. |
| Incorrect refund amount | Contact the support team with your transaction ID for further assistance. |

## 5. Notification Issues

| Issue | Possible Solution |
|-------|-------------------|
| No email notifications | Verify that your registered email address is correct and check your spam folder. |
| Missing payment notifications | Refresh the application and verify your notification preferences in **Account Settings**. |
| Delayed notifications | Notifications may be delayed due to network or email service issues. Wait a few minutes and check again. |

## 6. Browser and Connectivity Issues

| Issue | Possible Solution |
|-------|-------------------|
| Application not loading | Refresh the page or clear your browser cache. |
| Unsupported browser | Use the latest version of Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari. |
| Slow performance | Check your internet connection and close unnecessary browser tabs or applications. |
| Connection timeout | Verify your network connection and try again after a few minutes. |
